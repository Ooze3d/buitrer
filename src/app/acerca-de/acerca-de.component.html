Acerca del proyecto Buitrer

Nombre del alumno: Jose Antonio García López

Curso: 2ºDAW (Grupo 1)

Herramientas usadas:

  • Visual Studio Code
  • Angular 11 (Toda la funcionalidad front-end y las llamadas al servidor)
  • Google Firebase (Almacenamiento de datos)
  • Bootstrap 4 (Styling)

Funcionalidades de Angular usadas:

  • String interpolation: Para todas las llamadas a propiedades, atributos y variables
  • Pipes: Para el día y la hora en la lista de buits
  • Property binding: Para anular el botón "Buitréalo!" cuando el contenido del buit está vacío
  • Event binding: Todas las llamadas a click o input del formulario
  • Two way databinding: Todos los [(ngModel)] en los inputs de todos los formularios
  • Directivas: *ngIf en el menú, *ngFor en la lista de buits y ngSwitch en la barra izquierda
  • Binding de padre a hijo (@Input): Lista de buits y paso de usuario a componente Menu
  • Binding de hijo a padre: Logout, Login, Sección (para definir lo que mostramos con ngSwitch)
  • Servicios: Objeto "nuevo buit" y lista de buits global para toda la aplicación
  • Routing: Para esta ventana de dialogo (el resto de las secciones requerían data binding y se complicaba el routing)
  • Validación de formularios: Control de campos vacíos y duplicado de usuarios
  • Base de datos: Almacenamiento de usuarios y lista de buits en Firebase

Bibliografía:

  • Apuntes del módulo Tanto Angular como Bootstrap (módulo de Diseño de Interfaces)
  • Curso "Angular, The Complete Guide", impartido por Maximilian Schwarzmüller en Udemy
